## 🚀 Featured Project

### 🔐 [Pashupatastra](https://github.com/intrudx/Pashupatastra) — Enterprise Security & AI Orchestration Suite
`Python` · `FastAPI` · `asyncio` · `OpenAI` · `Claude` · `Gemini` · `Docker` · `UFW`

A parent monorepo housing two enterprise-grade security tools built from the ground up:

- **Bhairav (EDR)** — Cross-platform Endpoint Detection & Response system. Monitors low-level kernel and network events, parses anomalies via custom heuristics, and routes threats through integrated AI analysis pipelines (OpenAI / Claude / local models).
- - **Shankar (Orchestration)** — Secure CLI engine and web dashboard. Spins up isolated Multipass VMs with localized Gemini LLM agents (OpenClaw), WebSocket root terminal, and strict 2FA boundaries.
 
  - Built with **Defense in Depth**: Fernet AES-encrypted secrets, UFW firewall isolation, asyncio/FastAPI concurrency, TOTP-gated access.
