A parent monorepo housing two enterprise-grade security tools:

- **Bhairav (EDR)** — Cross-platform Endpoint Detection & Response system. Monitors low-level kernel and network events, parses anomalies via custom heuristics and integrated AI threat analysis pipelines (OpenAI / Claude / local models).
